## Usage ##

Create a new project base:

```bash
git clone https://github.com/magsdk/boilerplate.git my-project
```

Remove git metadata, build and start services:

```bash
cd my-project
rm -rf .git
magsdk
# or
DEBUG=* magsdk
```
